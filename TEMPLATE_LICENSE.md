## Usage In This Project

The project does not include the full Tabler template package. To keep the school AI Fair project small and safe, the UI was manually adapted using Tabler-inspired dashboard structure, dark cards, responsive layout patterns, and Bootstrap-like component styling.

No paid template files were used. No commercial-only assets were used. No template package was installed with npm.
